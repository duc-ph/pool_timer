# Pool Shot Timer

A simple web app for timing pool shots.

## Usage

Open `index.html` in any browser.

### Controls

- **SPACE** - Start/stop timer
- **Extension button** - Add 60 seconds
- **Duration input** - Change default timer length

### Flow

1. Press SPACE to start
2. Press SPACE to stop
3. Press SPACE to start a new timer
4. Repeat
